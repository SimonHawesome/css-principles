3 Main components

Tweens

w/ TweenMax

Simple animation

How dare!

How can I does?!

Import into your Angular component

								
									...
									// Right under Angular imports
									declare var TweenMax: any;
									...

Tween it out with:

								
										TweenMax.to('#tweenBox', 2, { 
											marginLeft: '400px', 
											marginTop: '150px', 
											backgroundColor: 'yellow', 
											width: '400px',
											rotation: 720, 
											transformOrigin:"50% 50%",
											onComplete: () => {
												TweenMax.to('#tweenBox p', 0.5, {
													opacity: 1
												});
											}
										});

Bring it on back with:

								
										TweenMax.set('#tweenBox', { 
											marginLeft: '0px', 
											marginTop: '0px', 
											backgroundColor: 'white', 
											width: '150px',
											rotation: 0 
										});

										TweenMax.set('#tweenBox p', {
											opacity: 0
										});

Default CSS, who dis?

								
										.box {
											background-color: white;
											height: 150px;
											margin-left: 0px;
											width: 150px;
										}
										
										.box p {
											color: black;
											line-height: 150px;
											text-align: center;
											opacity: 0;
										}

Timelines

w/ TimelineMax

Sequential animations

How dare I!?!?!?!

How can I does?!

Import into your Angular component

						
							...
							// Right under Angular imports
							declare var TimelineMax: any;
							...

Chain it out with:

						
							let tl = new TimelineMax();

							tl.to('#timelineBox', 1, { 
								marginLeft: '400px',
								marginTop: '100px'
							 })
								.to('#timelineBox', 1, {
									rotation: 720,
									transformOrigin: "50% 50%",
								})
								.to('#timelineBox', 0.5, { backgroundColor: 'yellow' })
								.to('#timelineBox', 0.5, { width: '400px', height: '50px' })
								.to('#timelineBox', 1, {
									backgroundColor: 'blue',
									rotation: 360,
									marginLeft: '100px',
									onComplete: () => {
										TweenMax.to('#timelineBox p', 0.5, {
											color: 'white',
											opacity: 1,
											lineHeight: '50px'
										});
									}
								});

Bring it on back with:

						
							TweenMax.set('#timelineBox', {
								marginLeft: '0px',
								marginTop: '0px',
								backgroundColor: 'white',
								height: '150px',
								width: '150px',
								rotation: 0,
								transformOrigin: "50% 50%"
							});

							TweenMax.set('#timelineBox p', {
								opacity: 0,
								lineHeight: '150px'
							});

Default CSS, who dis?

						
							.box {
								background-color: white;
								height: 150px;
								margin-left: 0px;
								width: 150px;
							}
							
							.box p {
								color: black;
								line-height: 150px;
								text-align: center;
								opacity: 0;
							}
							
							.box p.fancy {
								font-family: "Comic Sans MS", cursive, sans-serif;
								font-size: 18px;
							}

Easing

Ease In

Ease Out

Bounce

How can I does?!

Import into your Angular component

						
							...
							// Right under Angular imports
							declare var Power2: any;
							declare var Bounce: any;
							...

Take it ease-y with:

						
							TweenMax.to('#easeBox1', 1, {
								marginLeft: '600px',
								ease: Power2.easeIn
							});
				
							TweenMax.to('#easeBox2', 1, {
								marginLeft: '600px',
								ease: Power2.easeOut
							});
				
							TweenMax.to('#easeBox3', 1, {
								marginLeft: '600px',
								ease: Bounce.easeOut
							});
