How can I does?!

Import into your Angular component

								
									...
									// Right under Angular imports
									declare var TweenMax: any;
									...

Tween it out with:

								
										TweenMax.to('#tweenBox', 2, { 
											marginLeft: '400px', 
											marginTop: '150px', 
											backgroundColor: 'yellow', 
											width: '400px',
											rotation: 720, 
											transformOrigin:"50% 50%",
											onComplete: () => {
												TweenMax.to('#tweenBox p', 0.5, {
													opacity: 1
												});
											}
										});

Bring it on back with:

								
										TweenMax.set('#tweenBox', { 
											marginLeft: '0px', 
											marginTop: '0px', 
											backgroundColor: 'white', 
											width: '150px',
											rotation: 0 
										});

										TweenMax.set('#tweenBox p', {
											opacity: 0
										});

Default CSS, who dis?

								
										.tweenBox {
											background-color: white;
											height: 150px;
											margin-left: 0px;
											width: 150px;
										}
										
										.tweenBox p {
											color: black;
											line-height: 150px;
											text-align: center;
											opacity: 0;
										}
